# Mobile Sales Dashboard - Power BI

An interactive **Mobile Sales Dashboard** created in Microsoft Power BI to analyze sales performance, customer behavior, and key business metrics for the mobile phone industry.

## Project Overview
This dashboard provides deep insights into mobile sales data with powerful interactivity. Users can filter by **Brand**, **Mobile Model**, **Payment Method**, and explore trends across time, geography, and customer satisfaction.

### Key Metrics
- **Total Sales**: ₹769.20 Million
- **Total Quantity Sold**: 19,000 units
- **Total Transactions**: 4,000
- **Average Selling Price**: ₹40.11K

## Dashboard Features
- **Dynamic Slicers**: Brand, Mobile Model, Payment Method
- **Sales Trends**: By day of week, month, and city
- **Payment Analysis**: UPI, Debit Card, Credit Card, Cash distribution
- **Customer Ratings**: Good, Poor, and Average ratings breakdown
- **Top Performers**: iPhone SE, OnePlus Nord, Galaxy Note 20
- **Geographical Insights**: Sales distribution across major Indian cities (Delhi, Mumbai, Hyderabad, etc.)
- **Brand-wise Performance**: Apple, Samsung, OnePlus, Vivo, Xiaomi
- **Power BI Desktop**
- Interactive visualizations and bookmarks
